{\bfseries{Flat\+Buffers}} is a cross platform serialization library architected for maximum memory efficiency. It allows you to directly access serialized data without parsing/unpacking it first, while still having great forwards/backwards compatibility.

\doxysection*{Quick Start}


\begin{DoxyEnumerate}
\item Build the compiler for flatbuffers ({\ttfamily flatc})

Use {\ttfamily cmake} to create the build files for your platform and then perform the compilation (Linux example).


\begin{DoxyCode}{0}
\DoxyCodeLine{cmake\ -\/G\ "{}Unix\ Makefiles"{}}
\DoxyCodeLine{make\ -\/j}

\end{DoxyCode}

\item Define your flatbuffer schema ({\ttfamily .fbs})

Write the \href{https://flatbuffers.dev/flatbuffers_guide_writing_schema.html}{\texttt{ schema}} to define the data you want to serialize. See \href{https://github.com/google/flatbuffers/blob/master/samples/monster.fbs}{\texttt{ monster.\+fbs}} for an example.
\item Generate code for your language(s)

Use the {\ttfamily flatc} compiler to take your schema and generate language-\/specific code\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{./flatc\ -\/-\/cpp\ -\/-\/rust\ monster.fbs}

\end{DoxyCode}


Which generates {\ttfamily \doxylink{monster__generated_8h}{monster\+\_\+generated.\+h}} and {\ttfamily monster\+\_\+generated.\+rs} files.
\item Serialize data

Use the generated code, as well as the {\ttfamily \doxylink{classFlatBufferBuilder}{Flat\+Buffer\+Builder}} to construct your serialized buffer. (\href{https://github.com/google/flatbuffers/blob/master/samples/sample_binary.cpp\#L24-L56}{\texttt{ {\ttfamily C++} example}})
\item Transmit/store/save Buffer

Use your serialized buffer however you want. Send it to someone, save it for later, etc...
\item Read the data

Use the generated accessors to read the data from the serialized buffer.

It doesn\textquotesingle{}t need to be the same language/schema version, Flat\+Buffers ensures the data is readable across languages and schema versions. See the \href{https://github.com/google/flatbuffers/blob/master/samples/sample_binary.rs\#L92-L106}{\texttt{ {\ttfamily Rust} example}} reading the data written by {\ttfamily C++}.
\end{DoxyEnumerate}
