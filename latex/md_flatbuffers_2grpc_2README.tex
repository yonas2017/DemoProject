\chapter{GRPC implementation and test}
\hypertarget{md_flatbuffers_2grpc_2README}{}\label{md_flatbuffers_2grpc_2README}\index{GRPC implementation and test@{GRPC implementation and test}}
NOTE\+: files in {\ttfamily src/} are shared with the GRPC project, and maintained there (any changes should be submitted to GRPC instead). These files are copied from GRPC, and work with both the Protobuf and Flat\+Buffers code generator.

{\ttfamily tests/} contains a GRPC specific test, you need to have built and installed the GRPC libraries for this to compile. This test will build using the {\ttfamily FLATBUFFERS\+\_\+\+BUILD\+\_\+\+GRPCTEST} option to the main Flat\+Buffers CMake project.

\doxysection*{Building Flatbuffers with g\+RPC}

\doxysubsection*{Linux}


\begin{DoxyEnumerate}
\item Download, build and install g\+RPC. See \href{https://github.com/grpc/grpc/tree/master/src/cpp}{\texttt{ instructions}}.
\begin{DoxyItemize}
\item Lets say your g\+RPC clone is at {\ttfamily /your/path/to/grpc\+\_\+repo}.
\item Install g\+RPC in a custom directory by running {\ttfamily make install prefix=/your/path/to/grpc\+\_\+repo/install}.
\end{DoxyItemize}
\item {\ttfamily export GRPC\+\_\+\+INSTALL\+\_\+\+PATH=/your/path/to/grpc\+\_\+repo/install}
\item {\ttfamily export PROTOBUF\+\_\+\+DOWNLOAD\+\_\+\+PATH=/your/path/to/grpc\+\_\+repo/third\+\_\+party/protobuf}
\item {\ttfamily mkdir build ; cd build}
\item {\ttfamily cmake -\/DFLATBUFFERS\+\_\+\+BUILD\+\_\+\+GRPCTEST=ON -\/DGRPC\+\_\+\+INSTALL\+\_\+\+PATH=\$\{GRPC\+\_\+\+INSTALL\+\_\+\+PATH\} -\/DPROTOBUF\+\_\+\+DOWNLOAD\+\_\+\+PATH=\$\{PROTOBUF\+\_\+\+DOWNLOAD\+\_\+\+PATH\} ..}
\item {\ttfamily make}
\end{DoxyEnumerate}
