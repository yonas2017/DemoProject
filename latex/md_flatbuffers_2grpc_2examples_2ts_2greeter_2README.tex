The following is an example on how to run the TS grpc server. Make sure that you have {\ttfamily Typescript} installed

you would need to run {\ttfamily npm run build} or simply use {\ttfamily npm install \&\& tsc}

\doxysection*{How to run Server\+:}


\begin{DoxyItemize}
\item {\ttfamily npm run server}
\end{DoxyItemize}

\doxysection*{How to run Client\+:}


\begin{DoxyItemize}
\item {\ttfamily npm run client 3000} 
\end{DoxyItemize}
