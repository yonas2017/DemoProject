\doxysubsection{Detailed Description}
Helper class that builds Flex\+Buffers 

This class presents all necessary APIs to create Flex\+Buffers. A {\ttfamily Byte\+Buffer} will be used to store the data. It can be created internally, or passed down in the constructor.

There are some limitations when compared to original implementation in C++. Most notably\+: 
\begin{DoxyItemize}
\item 

No support for mutations (might change in the future).


\item 

Buffer size limited to {\ttfamily \doxylink{}{Integer\#\+MAX\+\_\+\+VALUE}}


\item 

Since Java does not support unsigned type, all unsigned operations accepts an immediate higher representation of similar type.


\end{DoxyItemize}
