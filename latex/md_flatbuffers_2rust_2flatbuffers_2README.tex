\chapter{Flatbuffers}
\hypertarget{md_flatbuffers_2rust_2flatbuffers_2README}{}\label{md_flatbuffers_2rust_2flatbuffers_2README}\index{Flatbuffers@{Flatbuffers}}
Flat\+Buffers is a cross platform serialization library architected for maximum memory efficiency. It allows you to directly access serialized data without parsing/unpacking it first, while still having great forwards/backwards compatibility. It was originally created at \doxylink{namespaceGoogle}{Google} for game development and other performance-\/critical applications.
