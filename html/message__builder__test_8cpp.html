void test_only_hack_update_fbb_reference ( MyGame::Example::MonsterBuilder &  monsterBuilder,
flatbuffers::grpc::MessageBuilder &  mb 
)

This function is a hack to update the FlatBufferBuilder reference (fbb_) in the MonsterBuilder object. This function will break if fbb_ is not the first member in MonsterBuilder. In that case, some offset must be added. This function is used exclusively for testing correctness of move operations between FlatBufferBuilders. If MonsterBuilder had a fbb_ pointer, this hack would be unnecessary. That involves a code-generator change though.
